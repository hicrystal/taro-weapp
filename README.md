

> 使用taro框架强力驱动

----------

### 简介

主要帮助scrum master 评估点数和任务难度

### 功能


进入房间=》 选定身份=》出牌=》统计点数=》下一个ticket


### 后端

技术：node + mysql + express 
地址：https://github.com/YTU94/taro-GGameing-server

### 使用

```
// 安装依赖
yarn
// 启动编译微信小程序（其他小程序请看package.json）
yarn dev:weapp
// 打包微信小程序（其他小程序请看package.json）
yarn build:weapp
```
